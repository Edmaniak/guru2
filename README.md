# guru2

A Nuxt 4.2 + Vue 3 + TypeScript project.

## Setup

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- **Nuxt 4.2** - The Vue Framework for modern web applications
- **Vue 3** - Progressive JavaScript Framework
- **TypeScript** - Typed superset of JavaScript
